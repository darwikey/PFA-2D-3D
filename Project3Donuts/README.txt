Project3Donut
=============

Des notices d'installation ou d'utilisation sont fournies dans ce 
dossier.

Ce logiciel utilise les outils suivants :
	- CMake, version requise minimum 2.8
	- Qt, version requise minimum 5.3.1
	- OpenGL, version requise minimum ES 2.0

Il est compatible avec les systèmes d'exploitation suivants :
	- Windows Seven 64bits
	- Windows 8.1 64 bits

	- Debian Jessie 64bits
	- Fedora 21 64bits
	
La machine doit avoir une carte graphique récente (NVIDIA ou 
équivalent) ou un chipset intégré (Intel HD Graphics 4000 ou 
équivalent).

Le dossier "ressources" doit être placé dans le dossier build après
compilation pour pouvoir exécuter le logiciel.
